
# ML Projects — House Price Analysis

A complete machine learning project applying 12 models to the 
Kaggle House Prices dataset.

## What's inside
- Data cleaning — handling 19 columns of missing values
- Feature engineering — label encoding, price categorization
- 6 Regression models compared (Linear, Ridge, Lasso, 
  Decision Tree, Random Forest, KNN, SVR)
- 6 Classification models compared (Logistic, Decision Tree,
  Random Forest, KNN, SVM, Naive Bayes)
- Full metric comparison — RMSE, R², Accuracy, F1 Score

## Results
| Task | Best Model | Score |
|------|-----------|-------|
| Regression | Random Forest | R² = 0.8953 |
| Classification | Random Forest | Accuracy = 94.86% |

## Dataset
Kaggle — House Prices: Advanced Regression Techniques
